# Mis Tesoros con Blazor Server

- ASP.Net Core
- Blazor Server
- Entity Framework
- Identity
- Google Maps

## Arrancar Proyecto

1. Descargar e instalar dotnet core <https://dotnet.microsoft.com/download>
2. dotnet restore
3. dotnet ef database update
4. ir a Pages/_Host.cshtml y agregar el api key de google maps
5. dotnet run
